Olá

'O que esta página é exatamente?', você pode estar se perguntando.
Bem em resumo, é uma interface para uma REST API.
Dando um pouco mais de detalhes, interagindo com essa página você pode adicionar e visualizar clocks, tudo isso por meio de recursos REST.

Para utilizar o sistema basta navegar pelas abas.
Na segunda aba você pode adicionar novos clocks ao sistema, e na terceira pode ver uma lista de todos os clocks existentes no sistema.

Adicionar

Para adicionar um novo clock ao sistema, basta informar a hora desejada e clicar em 'ADICIONAR'.

adicionar

Lista

Aqui você pode acessar a lista de todos os clocks registrados no sistema.

Se já existirem clocks no sistema, eles serão listados automaticamente

Caso tenha adicionado novos clocks, por favor utilizar o botão abaixo para atualizar a lista.
Atualizar a página também funciona.
